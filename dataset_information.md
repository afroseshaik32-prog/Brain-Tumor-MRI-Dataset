
# Dataset Information

## Source of Data

The Brain Tumor MRI Dataset was obtained from Kaggle, a trusted platform for machine learning datasets. It contains labeled MRI brain images categorized into four classes: Glioma, Meningioma, Pituitary, and No Tumor.

## Number of Records

Total Images: Approximately 5,712

## Class Distribution

- Glioma: 1321
- Meningioma: 1339
- Pituitary: 1457
- No Tumor: 1595

## Features

- MRI Brain Images
- Resolution: 224 × 224 Pixels
- RGB Color Images
- Four Output Classes

## Train-Test Split

- Training Data: 80%
- Testing Data: 20%

## Data Format

- JPG
- JPEG
- PNG
