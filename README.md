## Project Overview

This repository contains the dataset documentation for a Brain Tumor Detection system developed using a Convolutional Neural Network (CNN). The project aims to classify MRI brain images into four categories: Glioma, Meningioma, Pituitary, and No Tumor. The dataset is used for training and evaluating the deep learning model.

---

## Dataset Source

The dataset used in this project is the **Brain Tumor MRI Dataset**, obtained from **Kaggle**.

- **Dataset Name:** Brain Tumor MRI Dataset
- **Source Platform:** Kaggle
- **Dataset Link:** https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset
